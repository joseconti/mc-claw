## Competitive Analysis

### Steps
1. Map competitive landscape — see `references/landscape-mapping.md`
2. Build feature comparison matrix
3. Analyze positioning — see `references/positioning-frameworks.md`
4. Identify market trends and strategic responses

### Positioning Statement Template
For [target customer] who [need/problem], [Product] is a [category] that [key benefit]. Unlike [competitor/alternative], [Product] [key differentiator].

### Feature Rating Scale
- **Strong**: Market-leading capability
- **Adequate**: Functional, not differentiated
- **Weak**: Limited, significant gaps
- **Absent**: Does not have this capability
