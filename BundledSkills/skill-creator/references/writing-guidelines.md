# Skill Writing Guidelines

## Description Optimization
The description field is the primary mechanism for skill triggering. It should:
- Clearly state when to use the skill
- Include common trigger words and phrases
- Mention what NOT to use it for (reduce false triggers)

## Writing Style
- Explain WHY things are important instead of heavy-handed MUSTs
- Use theory of mind — make the skill general
- Start with a draft, review with fresh eyes, improve
- Keep SKILL.md focused, put details in references/

## Testing
- Create test prompts that should trigger the skill
- Create test prompts that should NOT trigger the skill
- Verify the AI follows the procedure correctly
- Check that references are consulted when needed
