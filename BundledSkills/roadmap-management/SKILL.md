## Roadmap Management

### Frameworks
- **Now / Next / Later**: Committed → Planned → Directional
- **Quarterly Themes**: 2-3 themes per quarter mapping to OKRs

### Prioritization — see `references/prioritization.md`
- RICE = (Reach x Impact x Confidence) / Effort
- MoSCoW: Must / Should / Could / Won't
- ICE = Impact x Confidence x Ease (1-10 each)
- Value vs Effort Matrix

### Capacity Planning
- 70% planned features
- 20% technical health
- 10% unplanned buffer

See `references/communication.md` for change communication.
