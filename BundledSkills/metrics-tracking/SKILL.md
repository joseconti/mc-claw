## Product Metrics

### Hierarchy
- **North Star**: Single metric capturing core value
- **L1 (Health)**: Acquisition, Activation, Engagement, Retention, Monetization, Satisfaction
- **L2 (Diagnostic)**: Funnels, feature usage, segments, performance

### Review Cadences
- Weekly (15-30 min): Catch issues
- Monthly (30-60 min): Deeper analysis
- Quarterly (60-90 min): Strategic assessment

See `references/goal-setting.md` for OKRs and target-setting frameworks.
See `references/dashboard-design.md` for dashboard principles.
