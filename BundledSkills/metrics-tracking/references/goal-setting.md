# Goal Setting Frameworks

## OKRs (Objectives and Key Results)
- Objectives: Qualitative, aspirational, time-bound
- Key Results: Specific, measurable, outcome-based, 2-4 per Objective

## Setting Metric Targets
- Baseline: Current performance
- Benchmark: Industry/competitor data
- Trajectory: Historical growth rate
- Effort: What resources are available
- Confidence: How certain are we

## L1 Metrics Detail
- Acquisition: New signups, signup conversion rate
- Activation: Activation rate, time to activate
- Engagement: DAU/WAU/MAU, stickiness, core action frequency
- Retention: D1, D7, D30, cohort curves, churn rate
- Monetization: Conversion, MRR/ARR, ARPU, net revenue retention
- Satisfaction: NPS, CSAT, support tickets
