# Creating Presentations with pptxgenjs

Install: `npm install -g pptxgenjs`

Use pptxgenjs to create .pptx files programmatically. Refer to the pptxgenjs documentation for the full API.

Key concepts:
- Create slides with `pres.addSlide()`
- Add text, images, tables, charts
- Set master slides for consistent layouts
- Export with `pres.writeFile()`
